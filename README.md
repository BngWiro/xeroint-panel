# Xeroint Panel Host

Xeroint Panel Host adalah panel manajemen server yang dirancang untuk memudahkan pengelolaan berbagai jenis *container* dan layanan. Awalnya dikembangkan untuk kebutuhan pribadi, proyek ini kini dibagikan secara publik agar dapat dimanfaatkan oleh komunitas secara luas.

## Dukungan Container

Saat ini, Xeroint Panel Host mendukung berbagai *environment* berikut:

- **SA-MP / OpenMP Server**
- **Node.js**
- **Python Server**
- **Website Host Server**
- **Minecraft Server**
